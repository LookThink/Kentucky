Kentucky

A leaping point for front-end development builds.

View on GitHub Download .zip Download .tar.gz

Patterns and grid systems are nice but not all projects are the same.

Kentucky aims to provide developers with a set of tools upon which their site can grow. It creates minimal CSS while providing access to high-powered, commonly requested mixins and variables to help jump start the development process.
